inbox.photo simplifies click and collect ordering of personalized photo products.

== Description ==

inbox.photo provides a simple way to customize and order personalized products such as prints, calendars, photobooks, canvas, collages, gifts, t-shirts, mugs and much more. inbox.photo enables set up of simple click and collect web stores.

This plug-in is designed to help you link your WordPress web site to your inbox.photo based web shop by providing shortcodes, snippets and sidebar widgets for your photo products customization ordering pages.

The inbox.photo design tool is fully responsive.

The inbox.photo editor includes:
* HTML5 based product editor
* Graphical back-office for easy definition of products
* Facebook, Google Photos, Instagram integration
* Google Fonts

If connected via the inbox.photo API (API token is required), additional order status information and notifications are also available.

[Register for free to inbox.photo](https://inbox.photo/start/) and get 500 free credits for set-up and testing.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/inboxphoto` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the "Plugins" screen in WordPress.
1. Use the Settings -> inbox.photo screen to configure the plugin.

Usage is documented on the Settings page of the plug-in.

== Frequently Asked Questions ==

= Is inbox.photo free? =

Set-up is free. We only charge for products ordered according to this [price list](https://inbox.photo/pricing/). This said, when creating your account, you get 500 free credits for set-up and testing.

= How can I test the plug-in runs effectively? =

The generated buttons, snippets and sidebar widgets should open your inbox.photo web shop in the product customization page.

= Do I need access to the inbox.photo API? =

No. Connection to the inbox.photo API only provides additional order status information and notifications. All other functions operate without requiring a connection to the inbox.photo API.
Contact us for pricing information.

= What does the option to sync customer account information to WordPress? =

This option automatically updates your WordPress instance with customer information from inbox.photo. This can be useful if you manage your newsletters from WordPress. Customers are added with the Customers role.

= What does the inbox.photo Premium Pack offer? =

The inbox.photo Premium Pack includes our Resource Manager allowing you to create your own templates for your products.
The inbox.photo Premium Pack also includes access to our API allowing for building any kind of applications to manage customer information and orders.
More features will come in the future.

= inbox.photo does not show up in my iframe =

inbox.photo needs to be cross origin enabled. Please contact us to add your domain to the cross origin list.

= Does inbox.photo provide advanced functions such as shipment? =

No. inbox.photo is really designed for in-shop pick-up. For more complete ecommerce functions, you should consider Creator Five. It features a [plug-in for WooCommerce](https://wordpress.org/plugins/creator-five-for-woocommerce/).
